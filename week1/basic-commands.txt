cd: change directory
ls: list content of the directory (-a to view hidden files)
pwd: present working directory
touch: create file
cat: read file
nano/vim: text editors to create/edit files
cp: copy a file
rm: remove a file(-rf to remove a directory)
mkdir: create directory
mv: move files/ rename files
